### 功能概述
- **首页**：管理后台入口
- **游戏管理**：游戏列表管理
- **商品管理**
  - 供货商管理：列表展示、搜索筛选、新增/编辑
  - 货源管理：按供货商分组展示、商品折扣与折扣时间（部分商品无活动推广时为空）
  - 活动推广：活动名称（新增可修改/编辑时只读）、活动ID（系统自增）、供应商单选下拉 + 货源多选下拉（带复选框）、活动时间精确到小时、折扣输入、新增/编辑/启用/禁用、表格筛选（供应商/活动时间/状态）、活动状态按时间自动计算（待执行/执行中/已完成/禁用/已暂停/已过期）、刷新按钮
- 侧边栏菜单切换
- 页面内 toast 提示（替代浏览器 alert）
- 时间输入框统一样式（text 类型 + placeholder + readonly + ~ 分隔符）

### 技术实现
- 纯静态 HTML + Vanilla JS + 内联 CSS
- Ant Design 视觉风格（#001529 侧边栏、#1890ff 主色）
- 自定义下拉组件（单选/多选带复选框）
- GitHub Pages 部署

### 修改内容
- 初始版本，从零构建雷神商城管理后台
- 活动推广：活动名称+活动ID、6种动态状态计算、11条 mock 数据覆盖全部状态
- 所有数据为 mock 数据
